# AIproject : 비판적 재평가 텍스트를 통한 planner Model 성능 개선 듀얼 시스템 VLA 모델 개발
# 프로젝트 개요
OpenVLA와 같은 모델은 다양한 작업에 대해 행동 토큰을 생성할 수 있어 실제 로봇 제어에 활용할 수 있습니다. 그러나 VLA모델은 행동을 생성하는데 집중되어 있으며, 생성된 행동이 적절한지 스스로 평가하거나 수정하는 과정이 부족하다고 느껴 비판적 텍스트를 통해 로봇이 스스로 더 학습하게 하는 연구를 진행하였습니다. 실제 로봇 환경 구축의 비용과 제약을 고려하여 LIBERO 시뮬레이션을 환경에서 실험을 수행하여 강화학습을 통해 행동 수정 정책을 학습하였습니다.
# 연구 배경
기존의 로봇 제어 모델은 행동의 적절성에 대한 재검토 과정이 존재하지 않음
